# 更新日志

## v1.3.7 (2026-05-15)

### 🏗️ 前端架构拆分
- 单文件 `index.html` (2285行) 拆分为 3 个 JS 模块：
  - `static/js/app.js` (292行) — 状态管理 / 工具函数 / 导航 / Dashboard / Modal
  - `static/js/auth.js` (153行) — 登录 / 注册 / 登出 / Session / 个人信息
  - `static/js/products.js` (1357行) — 产品 / 报价 / 导入 / 管理面板
- `index.html` 精简至 480 行，仅保留 HTML 结构和 CSS

### 🔍 搜索统一增强
- 全部 3 个搜索框统一支持：名称 / 规格 / 型号 / 功能 / 厂家 / 拼音 / 缩写
- 产品管理页搜索改为输入即搜（300ms→500ms 防抖），不再需要回车
- 选择产品对话框改为前端本地过滤 + 拼音匹配（无需等待后端）
- 后端 API 新增 `_py`（全拼）和 `_py_initials`（首字母）字段

### ⌨️ IME 输入法适配
- 添加 `compositionstart` / `compositionend` 检测
- IME 组字过程中暂停搜索，选字完成后自动触发
- 解决拼音输入中途页面刷新导致输入错误的问题

### 🐛 Bug 修复
- 修复 `searchDelay` 函数缺少结尾 `}` 导致整页 JS 崩溃（页面空白）
- 修复 `add_pinyin_field` 和 `pinyin_match` 中 `p[0]` vs `p[0][0]` 索引错误
  - 全拼字段误用首字母，导致 `hongwai` 等全拼搜索返回 0 结果
- 修复选择产品对话框中分类 + 搜索的组合过滤逻辑

### ⚡ 性能优化
- 选择产品对话框恢复本地过滤（392 条产品毫秒级响应）
- 搜索防抖统一调整为 500ms，减少后端请求频率
